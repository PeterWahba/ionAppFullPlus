Instructions to import this component on your ionic app

Plugins and dependencies needed:
ionic cordova plugin add cordova-plugin-media-capture
ionic cordova plugin add cordova-plugin-media
ionic cordova plugin add cordova-plugin-file
 
# Install NPM Packages
npm install --save @ionic-native/media-capture @ionic-native/media @ionic-native/file

Also if you want to keep the colors, you should import the them/variables.scss file which contain the colors
